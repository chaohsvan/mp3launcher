# 复古 MP3 风格音乐启动器

这是一款以复古 MP3 播放器为灵感、以音乐控制为核心的极简安卓启动器。
![fig1](/tempshow/fig1.jpg)
![fig2](/tempshow/fig2.jpg)


## 音乐为中心的主屏
*   深度整合媒体播放信息。
*   显示像素化专辑封面、歌曲信息及播放状态。

## 创新的音量键交互
*   **短按** 音量+/-
    *   切换上一首 / 下一首歌曲。
*   **长按** 音量+/-
    *   切换播放 / 暂停状态。

## 高效的应用管理
*   提供包含所有应用的应用抽屉。
*   支持长按应用图标进行**置顶/取消置顶**操作。
*   支持通过顶部的 **快速滚动条（Top Scroller）** 上下滑动来快速导航应用列表。
*   内置应用搜索功能，点击空白区域可自动隐藏键盘和搜索框。

## 沉浸式界面
*   隐藏系统状态栏和导航栏。
*   以简约风格显示电量和音量信息。

## 媒体通知控制
*   需要获取“通知访问”权限，以读取和控制其他音乐 App（如 Spotify）的播放。
